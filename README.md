# 用javascript实现常用数据结构和算法

## 队列
### 普通顺序队列

> 先进先出

```javascript
const sqQueue = new sqQueue()
```

### 定长顺序队列

> 达到固定长度后丢弃队尾

```javascript
const fixedLengthSqQueue = new fixedLengthSqQueue(5)
```

## 栈
### 普通顺序栈

> 先进后出

```javascript
const sqStack = new SqStack()
```

## 链表
### 普通单链表

> 顺序查找 随机存储
